3 VOICE AI ENGINE

Motor de Conversatie Vocala

Apeluri telefonice gestionate complet de AI, cu voce naturala, transcriere in timp real si intelegere contextuala a limbii romane.

Voce naturala, conversatie reala

Sistemul utilizeaza OpenAI Realtime API pentru conversatii vocale cu latenta minima, combinat cu Twilio pentru telefonie de inalta calitate.

  • ✓Streaming audio bidirectional — audio in timp real prin WebSocket cu codec g711 (standard telefonie)
  • ✓6 voci native + voce clonata — personalizare completa cu ElevenLabs voice cloning
  • ✓Barge-in natural — utilizatorul poate intrerupe botul, exact ca intr-o conversatie reala
  • ✓VAD semantic — detectie inteligenta a vorbirii (nu doar pe volum, ci pe sens)
  • ✓Transcriere Whisper — cu filtru anti-halucinare pentru limba romana
  • ✓Analiza sentiment post-apel — scor de la -1.0 la +1.0 cu clasificare automata

4 KNOWLEDGE RAG

Baza de Cunostinte Inteligenta

Sistem avansat de Retrieval-Augmented Generation cu cautare hibrida vector + full-text, optimizat pentru limba romana.

1
Ingestie
Upload documente (PDF, DOCX, CSV, URL) sau conectori automatizati (WooCommerce, WordPress)
2
Chunking
Segmentare inteligenta cu respectarea paragrafelor, overlap configurabil per sursa (128-512 tokens)
3
Embedding
Vectorizare 1536 dim cu OpenAI text-embedding-3-small, batch de 100 cu rate limiting
4
Indexare
HNSW index (pgvector) pentru vectori + GIN index pentru full-text search cu tsvector romanian
5
Retrieval
Cautare hibrida RRF (Reciprocal Rank Fusion) + reranking conditionat + deduplicare
🔍
Cautare Hibrida RRF
Vector + Full-Text Search combinate
  • Cautare vectoriala cu HNSW (cosine similarity) — recall >95%
  • Full-text search cu suport nativ limba romana (stemming, stopwords)
  • Reciprocal Rank Fusion combina ambele score-uri cu FTS weight 1.5x
  • Pragul de similaritate minim: 0.55 (configurable per bot)
  • Suport 5 limbi: romana, engleza, germana, franceza, spaniola
🎯
Reranking Inteligent
Activare conditionata doar in zona incerta
  • Reranking LLM activat doar in zona incerta (similaritate 0.50-0.78)
  • Eliminat cand rezultatele sunt clare (>0.78) sau prea slabe (<0.50)
  • Deduplicare: max 3 chunk-uri per document sursa
  • Parent-child retrieval: include chunk-urile adiacente pentru context complet
  • Business-aware scoring: FAQ +8%, documente recente +5%, title match +4%
📚
Surse de Cunostinte
Multiple tipuri de ingestie
  • Upload manual: PDF, DOCX, TXT, CSV cu parsare inteligenta
  • Website scan: Crawl automat cu conversie HTML-to-Markdown
  • Conectori: WooCommerce, WordPress cu sync automat
  • Agenti AI: Generare automata de continut (FAQ, ghiduri)
  • Manual: Editare directa cu preview instant
⚡
Optimizari Performanta
Cache multi-nivel si fallback
  • Cache rezultate 5 minute, cache embeddings 24 ore
  • HNSW index cu ef_search=100 pentru recall optim
  • Fallback automat la FTS-only daca embedding-ul esueaza
  • Budget de context adaptat per canal (3500 tokens chat, 1600 voice)
  • Near-duplicate removal (>80% word overlap eliminat)

8 HUMAN-AI COLLABORATION

Escaladare Inteligenta

Trei cai de escaladare catre agentii umani, cu context complet transferat si programare callback integrata.

🙋
Escaladare la Cerere
Clientul cere explicit un operator
  • Detectie keywords: "operator", "om", "persoana reala", "ajutor"
  • HandoffRequest creat cu rezumatul conversatiei (20 mesaje)
  • Intent-urile detectate si produsele discutate sunt transferate
  • Botul intreaba metoda preferata de contact (email/telefon)
😤
Escaladare pe Frustrare
Detectie automata a nemultumirii
  • 6 semnale analizate: repetitii, limbaj negativ, CAPS, "!!!", cautari esuate, mesaje scurte dismissive
  • Scor ≥60 = HIGH: opreste vanzarea, ofera operator imediat
  • Scor 30-59 = MEDIUM: empatie, ton adaptat
  • Conversatii lungi nerezolvate (≥15 mesaje) trigger automat
📅
Callback Scheduling
Programare apel de revenire
  • Formular integrat in widget: nume, telefon, data, interval orar
  • 3 intervale: dimineata (8-12), dupa-amiaza (12-17), seara (17-20)
  • Tracking status: pending → confirmed → completed
  • Note interne per callback cu timestamp si autor
👤
Captura Lead Inteligenta
Timing adaptat pe engagement
  • Scor engagement: click produse (+10), mesaje 6+ (+10), handoff (+30)
  • Botul cere date de contact natural, nu intreruptiv
  • Pre-chat form optional (captura inainte de conversatie)
  • Extractie automata email/telefon din text conversatie

⚡
Queue Workers
Procesare asincrona pentru: embeddings, sync produse, analiza sentiment, rezumate, webhooks. Cu retry si backoff exponential.
🔄
Circuit Breaker
Monitorizare in timp real a provider-ilor AI. La >80% erori in 5 minute, switch automat pe provider alternativ. Zero downtime.
💾
Cache Multi-Nivel
Redis cache pentru: sesiuni, RAG results (5 min), embeddings (24h), plan limits (5 min), model pricing. Reducere 60%+ API calls.
🔧
Workers & Job Processing
Procesare distribuita pe cozi dedicate
  • Knowledge Queue: Ingestie documente, chunking, batch embeddings (100/batch), cu timeout 5 min si 3 retry-uri
  • Default Queue: Mesaje multi-canal (WhatsApp, Facebook, Instagram), backoff [10s, 30s, 120s]
  • Post-Call Jobs: Sentiment analysis (delay 15s), rezumat (delay 20s), webhook (delay 25s) — secvential pentru consistenta
  • Sync Jobs: WooCommerce product sync cu lock 30 min, progress tracking Redis, batch 50 produse
  • Concurrency Control: Cache locks pentru operatii duplicate, rate limiting 200 req/min pe OpenAI
🏗️
Infrastructura
Self-hosted, control complet
  • PostgreSQL 16 + pgvector: Baza de date cu suport vectorial nativ, HNSW indexes, FTS romanian
  • Redis 7: Cache, queue broker, session store, rate limiting counters, circuit breaker state
  • WebSocket Server: Dedicat pentru real-time events si broadcast per-tenant
  • Nginx + Traefik: Reverse proxy cu SSL termination, WebSocket upgrade, load balancing
  • Coolify PaaS: Self-hosted deployment platform, zero-downtime deploys, auto-restart

10 SECURITATE

Securitate Enterprise

5 niveluri de izolare date per client, protectie SSRF, guardrails AI anti-halucinare si cost control automat.

🔒
Izolare Multi-Tenant 5 Niveluri
Database (FK constraints) → ORM (global scopes pe 22+ modele) → Controller (verificare explicita) → Middleware (TenantAccess) → Aplicatie (sesiune atomica)
🛡️
Anti-Halucinare AI
Guardrails aplicate ULTIMELE in prompt (nu pot fi suprascrise). Botul raspunde STRICT din context. Nu inventeaza preturi, termene sau date.
🌐
Protectie SSRF
Blocheaza IP-uri private (RFC 1918), loopback, link-local, cloud metadata. Validare DNS inainte de access URL. Doar HTTP/HTTPS permis.
💳
Cost Control Automat
Limita zilnica per client ($5/zi default). Max 2 call-uri LLM per request. Max 8000 tokens per cerere. Buget per conversatie (15 centi).
🔑
Autentificare Multi-Layer
Sanctum tokens API. HMAC-SHA256 pentru Twilio + Meta + WooCommerce webhooks. Sesiuni chat cu token anti-hijack. RBAC cu 4 roluri.
⚡
Rate Limiting Granular
30 msg/min per IP pe chat. 10 req/min pe voice sessions. 5 callback/ora. 60 req/min API general. Protectie per-endpoint configurabila.
